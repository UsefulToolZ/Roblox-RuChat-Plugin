# Roblox-RuChat-Plugin
A simple GUI tool to change your Russian roblox account to another country location.
Note: if your browser blocking download go to downloads page and press download file anyway
# Launching on Android devices
1. Open Roblox RuChat.apk
2. In opened window select Roblox RuChat and press allow
3. Then if the admin right granting panel does not apper restart the app and press install
4. Done!
# Launching on Windows devices
1. Disable antivirus (This is necessary so that the program can connect to your Roblox client)
2. Launch Roblox RuChat v1.7.7.4.exe
3. Give it admin rights to search the roblox process
4. Done!
